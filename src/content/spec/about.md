# ✍ 为什么写博客？



很多时候：今天学会了，明天忘了。如果只是学习，很容易停留在"我好像学过"。但如果能够：学习  ->  理解  ->  实践  ->  总结  ->  分享



知识就真正属于自己了。

所以，这个博客不仅是写给别人看的。

更是写给未来的自己。

希望几年以后再回来，还能记得今天因为解决一个 Bug 而开心半天。
