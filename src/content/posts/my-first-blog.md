## 🐙 第一次接触 GitHub

![GitHub学习](public\images\first-blog2.png)

刚开始我只知道：

```bash
git clone
```

能够把项目下载到本地。

但真正下载一个项目后，我发现：

> 我会 Clone，却不会运行项目。

于是开始学习：

* README 的作用
* 什么是依赖
* package.json 是什么
* npm 与 pnpm 的区别
* Astro 是什么

慢慢理解了一个项目从下载到运行的完整流程。

## 💻 搭建博客

为了拥有自己的博客，我选择了 Fuwari 主题。

### 安装依赖

Python项目：        

```bash
pip install
```

前端项目：

```bash
pnpm install
```

虽然工具不同，但本质都是安装项目运行所需的依赖。

---

### 启动项目

```bash
pnpm dev
```

启动成功后访问：

```text
http://localhost:4321
```

第一次看到博客运行成功时，真的非常激动。

---
